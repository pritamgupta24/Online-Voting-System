# Online-Voting-System
A web-based Online Voting System built using HTML, CSS, JavaScript, PHP, and MySQL. This project allows voters to cast their votes securely and helps administrators manage elections efficiently through a user-friendly interface.

Developed a web-based voting platform using HTML, CSS, JavaScript, PHP, and MySQL that allows users to register, log in, and cast their votes securely. The system prevents duplicate voting and shows real-time results after voting ends.

Admins can manage voters, candidates, and monitor election data through a dedicated dashboard. The project focuses on providing a secure, user-friendly, and transparent voting process.

Key Features:

Voter registration and secure login

One-vote-per-user validation

Real-time result display

Admin panel for management and monitoring

Tech Stack: HTML, CSS, JavaScript, PHP, MySQL
